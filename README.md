A minimalist and customizable progressive web application for listening to online radio stations.

## Features

- Exquisite default station selection

- Customizable station list

- Recent station history

- Light and dark themes

- Sleep timer

- HLS support

## Tech stack

- Alpine.js

- Simple.css
