About

FmGeek is a minimalist and customizable progressive web application for listening to online radio stations. You can also install it as a browser app on your device.

If you need inspiration on what to listen to, you can check out PublicRadioFan and RadioBrowser. Please note that only HTTPS streams are supported.

For bug reports and feature requests, please use the issue tracker. For questions, ideas, and other topics, please check out the discussions.

Happy listening!
